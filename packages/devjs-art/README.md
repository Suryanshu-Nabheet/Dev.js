# Devjs ART

Devjs ART is a JavaScript library for drawing vector graphics using [Devjs](https://github.com/Suryanshu-Nabheet/dev.js/).

It provides declarative and devjsive bindings to the [ART library](https://github.com/sebmarkbage/art/).

Using the same declarative API you can render the output to either Canvas, SVG or VML (IE8).
