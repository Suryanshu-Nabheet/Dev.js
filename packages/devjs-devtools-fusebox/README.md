# devjs-native-fusebox

This package is private and not expected to become public anytime soon. Consider using [devjs-devtools-inline](https://github.com/Suryanshu-Nabheet/dev.js/tree/main/packages/devjs-devtools-inline) or [devjs-devtools-core](https://github.com/Suryanshu-Nabheet/dev.js/tree/main/packages/devjs-devtools-core).

## What is Fusebox?
"Fusebox" is the internal codename for the new Devjs Native debugger stack based on Chrome DevTools.
