# babel-plugin-devjs-compiler

Devjs Compiler is a compiler that optimizes Devjs applications, ensuring that only the minimal parts of components and hooks will re-render when state changes. The compiler also validates that components and hooks follow the Rules of Devjs.

This package contains the Devjs Compiler Babel plugin use in projects that make use of Babel. You can find instructions for using this plugin here: https://devjs.dev/learn/devjs-compiler
